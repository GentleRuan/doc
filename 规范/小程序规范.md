# js

1、 异常
```
try {
       // 业务代码
    } catch (error) {
       console.log(error)
}
````


2、相关属性放在同对象下

```
swiper: {
      key1: "属性1",
      list: [
        sub: "11"
      ]
    },
    tagProp: {
      tagList: []
    }
}
```

3、采用es6， 异步用async 、 await, 

4、常量const 、变量 let
